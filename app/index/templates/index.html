Welcome to the MYO-xIA application

Context:

This application was developed within the 80-Prime MYO-xIA project. The MYO-xIA project aims to collect data from patients with congenital myopathies in order to analyse them using explainable AI approaches. In the field of congenital myopathies research, there is an important need for digitization and standardization of patient data in order to use modern data analysis methods.
This application has three main objectives:

  • Gather and format patient data to be ready for automatic exploitation (patient histology-report form, image annotation, OCR)
  • Help the discovery of new phenotype-genotype-histology association for each congenital myopathy subtype (standard vocabulary creation tool, per gene and class histology feature statistics)
  • Provide tools for automatic diagnosis based on histology features (live prediction of congenital myopathy subtype in the histology-report form)

Functionalities:

  • Module 1: Standard Vocabulary Tool: A tools to create standard vocabulary tree. Used to create the histology standard vocabulary based on Institut de Myologie de Paris reports. This vocublary can be explored and modify to help format free-text histology report in a structured format.

  • Module 2: Report Digitization: a form to digitalize histology reports based on the histology standard vocabulary.

  • Module 3: Image Annotation: allows recording and annotation of histology images of tissues from patients with congenital myopathy. It is possible to delimit areas on the images and to associate them with labels..

  • Module 4: Statistics and Visualisation: a page to view most common histology features in congenital myopathies per gene and per disease subtype. Also show general graph about repartition of data by age, genes and diagnosis and a correlation matrix between standard vocubulary terms.

How to Use:

Statistics and standard vocabulary are viewable without an account. In order to perform Report digitization, Image annotation or modify the standard vocabulary, you will need an account. Please contact the email in the footer to request an account.
